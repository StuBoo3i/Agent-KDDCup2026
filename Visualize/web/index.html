Trace.json 说明

Prompts（输入给模型）
每一步里看 prompt_messages。其中 system 是系统提示词，user 是任务描述和 observation。
Assistant Message（模型输出）
看 raw_response。这是模型原始返回文本，再被解析成 thought/action/action_input。
Tool Message（工具结果）
看 observation。它是工具执行结果，并会在下一轮作为 user observation prompt 送回模型。
时间字段
started_at_utc / completed_at_utc / elapsed_seconds 是 step 时间。 顶层 e2e_elapsed_seconds 是任务总耗时。
